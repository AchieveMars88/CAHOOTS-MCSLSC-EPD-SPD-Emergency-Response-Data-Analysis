# Data Analysis Overlook

The following folder utilizes the "analysis.ipynb" notebook to statistically and visually assess the cleaned data across all agencies with associated weather data.

## Used Data (From ../cleaning/[...] File Path):
1. CAD_CLEAN.csv.gz (Compressed)
2. SPD_CLEAN.csv
3. MCSLC_CLEAN.csv

## Running the Notebook:

1. Save the data as formatted in the repository to a local directory (only include the cleaned data files).
2. Run the notebook. The cleaned data will/would appear where they're placed in the repository.

## Dependencies:
Using Python base 3.13.5, the following packages are used:

- pandas (dataframe manipulation)
- numpy/scipy.stats (statistical assessment)
- matplotlib/seaborn (visual assessment)
- datetime (date parsing) and regex (value formatting) standard library usages
