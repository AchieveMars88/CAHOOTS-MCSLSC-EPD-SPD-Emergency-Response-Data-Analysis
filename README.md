# CAHOOTS, MCSLC, EPD, & SPD Response Data Analysis - Extreme Weather Impact

This repository contains an analysis of raw and cleaned datasets concerning service calls and associated responses/assistance by various services in Eugene and Springfield, OR. The analysis, concerning the response services of CAHOOTS (Crisis Assistance Helping Out On The Streets), MCSLC (Mobile Crisis Services of Lane County), EPD (Eugene Police Department), and SPD (Springfield Police Department), compares call types, densities, response times, and resolution classifications between ice storm conditions and typical/expected weather.

1. "cleaning" folder: concerns the data preparation and formatting portion of the project, with associated notebooks, raw/clean data, and a separate README file.

2. "analysis" folder: concerns the statistical analysis portion of the project, with associated notebooks, stats, visuals, and a separate README file.
